# 🛒 MiniMarket POS

Sistema de Punto de Venta (POS) moderno y eficiente para la gestión integral de un minimarket. Esta aplicación web permite controlar categorías, productos, clientes, ventas y generar reportes visuales, todo desde una interfaz atractiva y responsive.

## 🚀 Tecnologías Utilizadas

- ⚛️ React — Biblioteca para construir interfaces de usuario modernas.
- 🎨 TailwindCSS — Framework CSS utility-first para diseño responsivo y personalizable.
- 🌼 DaisyUI — Componentes UI personalizables sobre TailwindCSS.

---

## 🧩 Módulos del Sistema

### 📁 Categorías
- Crear, editar, eliminar y listar categorías de productos.
- Asignación de productos a categorías.

### 📦 Productos
- CRUD completo de productos.
- Control de stock, precios y códigos de barras.
- Visualización rápida de productos por categoría.

### 👥 Clientes
- Registro y gestión de clientes frecuentes.
- Historial de compras por cliente.
- Búsqueda rápida por nombre, documento o teléfono.

### 💵 Ventas
- Generación de tickets o comprobantes.
- Selección de productos, cantidades y clientes.
- Cálculo automático de totales, impuestos y cambio.
- Registro de ventas y métodos de pago.

### 📊 Reportes
- Reportes diarios, semanales y mensuales de ventas.
- Visualización de productos más vendidos.
- Exportación a PDF o Excel (planificado).

## 🧪 Instalación y Ejecución

1. **Clona el repositorio**
```bash
git clone https://github.com/tuusuario/sys_minimarket_pos
cd sys_minimarket_pos
```

2. **Instala las dependencias**

```bash
npm install
```

3. **Ejecuta la aplicación en desarrollo**

```bash
npm run dev
```
